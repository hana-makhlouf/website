3D CAD Loader: Second Year Electronics Project

This project involved designing, programming and packaging a cross-platform CAD loader. The software was used to edit and view 3D shapes and was aimed at the aerospace industry.

This was a group project with 3 coursemates. I was personally responsible for the GUI design, which was done in Qt (front end). This was a C++ project using VTK.
